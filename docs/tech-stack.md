# Tech Stack

## Frontend: React, Vite, TypeScript, Tailwind CSS

**Why it was used:**

- React is a good fit for component-driven UI work.
- Vite keeps the development experience fast and lightweight.
- TypeScript adds structure and reduces avoidable frontend mistakes.
- Tailwind CSS helps keep styling consistent and maintainable.

## Backend: FastAPI, Python

**Why it was used:**

- FastAPI provides a clean API layer with strong request validation support.
- Python keeps the backend productive and easy to extend.
- The combination works well for a small-to-medium production application.

## Database: PostgreSQL

**Why it was used:**

- Reliable relational storage
- Good fit for orders, products, and operational records
- Strong ecosystem and deployment support

## Payments: Stripe

**Why it was used:**

- Trusted payment infrastructure
- Secure handling of sensitive payment data
- Easier integration for checkout flows than building payment logic directly

## Infrastructure: VPS Linux, Nginx, Cloudflare

**Why it was used:**

- VPS hosting gives control over the deployment environment
- Nginx is a proven reverse proxy and request router
- Cloudflare improves delivery, TLS handling, and edge protection

## Version Control: Git / GitHub

**Why it was used:**

- Standard collaboration workflow
- Clean history for product iteration
- Easy separation of private source and public case-study material
