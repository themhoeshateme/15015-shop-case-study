# Roadmap

These are sensible next steps for a project of this type:

## Product

- Improve product discovery and filtering
- Add stronger search and sorting options
- Expand merchandising support

## Admin

- Add more efficient management tooling
- Improve content editing workflows
- Add audit visibility for operational changes

## Checkout

- Broaden payment state handling and recovery paths
- Improve edge-case handling for abandoned or interrupted checkouts
- Add clearer post-payment status visibility

## Operations

- Add more monitoring and alerting
- Improve deployment automation
- Expand QA coverage for critical user flows

## Documentation

- Add non-sensitive release notes
- Add a sanitized onboarding guide for collaborators
- Keep public case-study assets current
